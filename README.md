A vanilla javascript "Shopping Cart" project, made with the intention of acquiring more experience on working with JavaScript and it's functionalities in a recurring enviroment, in this case, a furniture E-Commerce Website was chosen as the foundation for thre project.

## How to Run
Clone this repository:

    git clone https://github.com/PedroMilani04/pedro-portfolio.git
    
Have Node.js installed in your computer. With terminal opened in project folder, install the dependencies :

    npm install
        or
    yarn
    
Then type this command in to run local server:

    npm run dev
       or
    yarn dev
